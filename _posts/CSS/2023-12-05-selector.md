# 전체 선택자
{: .notice--warning .text-center}

```html
<style>
    * {
        font-weight: bold;
    }
</style>
```

# 아이디 선택자
{: .notice--warning .text-center}

```html
<!-- id는 하나의 요소에만 사용 가능함(1:1 매핑) -->
<style>
    #id1, #id2 {
        background-color: bisque;
    }
    #id1 {
        color: brown;
    }
    #id2 {
        font-size: large;
    }
</style>

<div id="id1">
    id1 영역
</div>
<div id="id2">
    id2 영역
</div>
```

# 클래스 선택자
{: .notice--warning .text-center}

```html
<!-- class는 여러개의 요소(그룹)에도 사용 가능함(1:n 매핑) -->
<style>
    /* classP를 부모로 둔 .classC 자식 요소만 선택 */
    .classP .classC {
        color: white;
        background-color: black;
    }
    /* classP를 부모로 둔 요소와 .classC를 부모로 둔 요소 전체 선택 */
    .classP, .classC {
        font-style: italic;
    }
</style>

<div class="classP">
    classP 영역
    <div class="classC">
            ClassP의 ClassC 자식 영역
    </div>
    <div>
        classP의 일반 자식 영역
    </div>
</div>
<div class="classC">
    단독 ClassC 영역
</div>
```

# 기본 속성 선택자
{: .notice--warning .text-center}

```html
<style>
    div[name=name1] {
        background-color: khaki;
    }
    .nameClass[name=name1] {
        text-align: center;
    }
</style>

<div name="name1">div name1 영역</div>
<div name="name2">div name2 영역</div>
<div class="nameClass" name="name1">div name1 nameClass 영역</div>
<div class="nameClass" name="name2">div name2 nameClass 영역</div>

<style>
    /* 선택자[속성~=값] : 특정 단어 포함 */
    input[type~=text] {
        text-align: center;
    }
    /* 선택자[속성^=값] : 시작 단어 포함 */
    input[type^=te] {
        font-size: 20px;
    }
    /* 선택자[속성$=값] : 끝나는 단어 포함 */
    input[type$=rd] {
        text-align: right;
    }
    /* 선택자[속성*=값] : 속성의 단어 포함 */
    input[type*=word] {
        height: 100px;
    }
    input[type=password] {
        color: red;
        background-color: bisque;
    }
</style>

<input type="text">
<input type="password">
```

# 후손/자식 선택자
{: .notice--warning .text-center}

```html
<style>
    /* test1의 모든 h4 후손 해당 */
    #test1 h4 {
        color: crimson;
        background-color: deepskyblue;
    }
    /* test1의 자식 h4만 해당 */
    #test1 > h4 {
        text-align: center;
    }
    /* test1의 자식 div의 자식 h4만 해당 */
    #test1 > div > h4 {
        text-align: right;
    }
</style>

<div id="test1">
    <h4>h4 자식영역1</h4>
    <h4>h4 자식영역2</h4>
    <h3>h3 자식영역</h3>
    <div id="test2">
        <h4>h4 후손영역</h4>
        <h3>h3 후손영역</h3>
        <p>p 후손영역</p>
        <div>div 후손영역</div>
    </div>
    <p>p 자식영역</p>
</div>
```

# 동위(형제) 선택자
{: .notice--warning .text-center}

```html
<style>
    /* 동위 관계에서 뒤에 위치한 선택자 태그를 선택 */
    #div-test1 + div {
        background-color: red;
    }
    /* 동위 관계에서 뒤의 선택자 태그의 뒤에 위치한 선택자 태그를 선택 */
    #div-test1 + div + div {
        background-color: orange;
    }
    /* 동위 관계에서 뒤에 위치한 선택자 태그 전체를 선택 */
    #div-test1 ~ li {
        font-weight: bold;
    }
</style>

<div id="div-test1">div test1</div>
<div id="div-test2">div test2</div>
<div id="div-test3">div test3</div>
<li id="div-test4">div test4</li>
<li id="div-test5">div test5</li>
<li id="div-test6">div test6</li>
```

# 상태 선택자
{: .notice--warning .text-center}

```html
<style>
    /* checked : checkbox 체크 상태 시 */
    input[type='checkbox']:checked {
        width: 30px;
        height: 30px;
    }
    /* focus : input 태그 선택 시 */
    #u_id:focus,
    #u_pw:focus {
        background-color: red;
        transition: all 1000ms;
    }
    /* disabled : 해당하는 옵션 선택 불가능 상태 */
    option:disabled {
        background-color: black;
    }
    /* enabled : 해당하는 옵션 선택 가능 상태 */
    option:enabled {
        background-color: green;
    }
</style>

<input type="checkbox" id="apple" name="fruits" value="사과">사과
<input type="checkbox" id="banana" name="fruits" value="바나나">바나나
<input type="checkbox" id="orange" name="fruits" value="오렌지">오렌지
<br>
아이디 : <input type="text" id="u_id">
비밀번호 : <input type="text" id="u_pw">
<br>
<select>
    <option value="10" disabled>10대</option>
    <option value="20">20대</option>
    <option value="30">30대</option>
    <option value="40">40대</option>
</select>
```

# 일반 구조 선택자
{: .notice--warning .text-center}

```html
<style>
    /* 형제 관계 태그 중 첫번째 태그 선택 */
    #test1 p:first-child {
        background-color: red;
    }
    /* 형제 관계 태그 중 마지막 태그 선택 */
    #test1 :last-child {
        background-color: yellow;
    }
    /* 형제 관계 태그 중 앞에서 수열번째 태그 선택 */
    /* 테스트2, 테스트4 선택됨 */
    #test1 p:nth-child(2n) {
        background-color: yellowgreen;
    }
    /* 형제 관계 태그 중 뒤에서 수열번째 태그 선택 */
    /* 테스트4, 테스트1 선택됨 */
    #test1 :nth-last-child(3n) {
        color: rgb(145, 255, 0);
    }
</style>

<div id="test1">
    <p>테스트1</p>
    <p>테스트2</p>
    <p>테스트3</p>
    <p>테스트4</p>
    <p>테스트5</p>
    <pre>테스트6</pre>
</div>
```

# 문자 선택자
{: .notice--warning .text-center}

```html
<style>
    /* 요소의 첫 글자 */
    #test3 p::first-letter {
        font-size: 2em;
    }
    /* 요소의 첫 라인 */
    #test3 p::first-line {
        background-color: yellowgreen;
    }
    /* after : 선택한 요소의 끝, content : 요소 추가 */
    #test3 p::after {
        content: "#### 끝 ####"
    }
    /* 선택한 요소를 기준으로 네이밍한 카운터(여기서는 rint)에 카운트함 */
    #test3 p {
        counter-increment: rint;
    }
    /* before : 선택한 요소의 처음, content : 요소 추가 */
    /* 영역1, 영역2 ====> 1.영역1, 2.영역2 로 요소가 나타남*/
    #test3 p::before {
        content: counter(rint) ". ";
    }
    /* selection : 선택(드래그)된 상태 */
    #test3 p::selection {
        background-color: red;
        color: white;
    }
</style>

<div id="test3">
    <p>영역1</p>
    <p>영역2</p>
</div>
```

# 선택자 우선순위
{: .notice--warning .text-center}

- 기본적으로 CSS 속성은 코드 순서대로(위 -> 아래) 적용
- 같은 태그(요소)에 여러 CSS 속성이 설정된 경우 우선순위에 따라 적용
1. !important 설정된 속성
2. 인라인 스타일
3. 아이디 선택자
4. 클래스 선택자
5. 태그 선택자

```html
<style>
    #test2,
    #test3 {
        /* test3이 !important에 의해 우선순위 밀림 */
        background-color: yellow;
    }
    div {
        width: 100px;
        height: 100px;
        /* 인라인 스타일, id에 의해 우선순위 밀림 */
        background-color: orange;
    }
    /* !important, 인라인 스타일, id에 의해 우선순위 밀림 */
    .test {
        background-color: purple;
    }
    /* !important, 인라인 스타일, id, class에 의해 우선순위 밀림 */
    p {
        background-color: red;
    }
    /* 우선순위 1순위 */
    p {
        background-color: violet !important;
    }
</style>

<h2>선택자 우선순위 테스트</h2>
<div id="test2" class="test">우선순위 테스트1</div>
<div id="test2" class="test" style="background-color: yellowgreen;">우선순위 테스트2</div>
<p id="test3" class="test">우선순위 테스트3</p>
<p id="test4" class="test">우선순위 테스트4</p>
```
