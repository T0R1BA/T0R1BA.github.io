# 박스 모델?
{: .notice--warning .text-center}

```lua
Margin
|-----------------------------------|
|   Border                          |
|   |---------------------------|   |
|   |   Padding                 |   |
|   |   |-------------------|   |   |
|   |   | Content           |   |   |
|   |   |                   |   |   |
|   |   |-------------------|   |   |
|   |---------------------------|   |
|-----------------------------------|
```

```css
.box {
    /* Content : 요소의 실제 내용이 위치하는 곳 */
    width: 200px;
    height: 100px;
    /* padding : 내부 여백으로, 테두리와 콘텐츠 사이의 간격을 조절 */
    padding: 20px;
    /* border : 요소의 테두리, 별개로 outline 지정할 수 있음 */
    border: 2px solid #333;
    /* margin : 외부 여백으로, 다른 요소와의 간격을 조절 */
    margin: 10px;
}
```
