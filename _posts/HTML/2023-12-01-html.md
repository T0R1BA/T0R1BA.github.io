# 기본 구조
{: .notice--warning .text-center}

```html
<!-- <!DOCTYPE >는 HTML 문서의 최상단에 위치하며 문서의 형식이나 유형을 웹 브라우저에게 알려주는 역할을 합니다. -->
<!-- html속성은 HTML5 선언 예시입니다 -->
<!DOCTYPE html>

<!-- <html>는 HTML 문서의 시작을 나타냅니다. -->
<!-- lang속성은 웹 페이지가 어떤 언어로 작성되었는지를 나타내는 메타데이터로 웹 검색 엔진은 사용자의 언어 설정 및 검색 쿼리에 따라 적합한 결과를 제공하기 때문에 이에 영향을 줍니다. -->
<html lang="ko">

<!-- <head>는 문서의 메타데이터를 정의합니다. 제목, 스타일 시트, 스크립트 등이 여기에 포함될 수 있습니다. -->
<head>

</head>

<!-- <body>는 웹 페이지의 본문을 정의합니다. 텍스트, 이미지, 링크 등이 여기에 포함됩니다. -->
<body>
    <!-- Semantic은 의미론적인, 의미에 관한 것을 나타냅니다. HTML에서 Semantic HTML은 문서의 구조와 의미를 더 명확하게 표현하는 데 중점을 둔 웹 페이지 작성 방법을 의미합니다. Semantic HTML을 사용하면 코드가 읽기 쉽고 이해하기 쉽게 되며, 검색 엔진이나 웹 페이지를 사용하는 다른 기기 및 기술이 문서를 더 잘 이해할 수 있게 됩니다. 다음은 Semantic HTML의 몇 가지 주요 태그들입니다. -->

    <!-- <header>는 웹 페이지나 섹션의 머리말을 정의하는 데 사용 -->
    <!-- 일반적으로 로고, 제목, 메뉴 등의 컨텐츠를 감싸는 용도로 사용되며, 웹 페이지 상단에 위치하는 부분을 나타냅니다. -->
    <header>
        <!-- <nav>는 네비게이션 링크를 정의합니다. 주로 메뉴나 링크 목록을 감싸는 용도로 사용됩니다. -->
        <nav>

        </nav>
    </header>

    <!-- <main>는 문서의 주요 콘텐츠를 정의합니다. 하나의 문서에는 하나의 <main> 요소만 있어야 합니다. -->
    <main>
        <!-- <section>는 문서의 섹션을 나타내며, 관련된 콘텐츠를 그룹화할 때 사용됩니다. -->
        <!-- 독립적으로 스타일링하거나 배포, 재사용하기에 용이합니다. -->
        <section>
        
        </section>

        <!-- 문서, 페이지, 애플리케이션, 또는 사이트에서 독립적으로 구분되거나 재사용 가능한 영역을 정의합니다. 일반적으로 뉴스 기사, 블로그 글, 포럼 게시물 등을 감싸는 데 사용됩니다. -->
        <article>

        </article>
    </main>

    <!-- <aside>는 주요 콘텐츠와 직접적인 연관성이 없는 콘텐츠를 나타냅니다. 주로 사이드바, 광고, 증강 정보 등을 감싸는 데 사용됩니다. -->
    <aside>
    
    </aside>

    <!-- <footer>는 웹 페이지나 섹션의 바닥글을 정의하는 데 사용됩니다. -->
    <!-- 주로 저작권 정보, 연락처 정보, 사이트 링크 등이 들어갑니다. -->
    <footer>
    
    </footer>
</body>

</html>
```

# 렌더링 방식: 블록과 인라인
{: .notice--warning .text-center}

- 블록(Block): 블록 요소는 항상 새로운 줄에서 시작하며, 가능한 최대 가로 너비를 차지합니다. 블록 요소는 부모 요소의 전체 너비를 차지하며, 수직으로 쌓이기 때문에 다른 요소들이 옆에 위치할 수 없습니다. 대표적인 블록 요소로는 div, p, h1부터 h6, form 등이 있습니다.

```html
<div>
  This is a block element. It takes up the full width available.
</div>
```

- 인라인(Inline): 인라인 요소는 새로운 줄에서 시작하지 않고, 요소가 끝날 때까지 수평으로 차지하는 너비만큼만 영역을 차지합니다. 다른 인라인 요소들과 나란히 나타날 수 있습니다. 대표적인 인라인 요소로는 span, a, strong, em, code 등이 있습니다

```html
<span>This is an inline element.</span>
```

## meta
{: .notice--success .text-center}

meta 태그는 문서의 메타데이터를 정의합니다. 메타데이터는 데이터에 대한 데이터로서, 문서의 특정 정보를 설명하거나 제공하는 역할을 합니다.

```html
<!-- 문자 인코딩(Charset 설정) -->
<!-- 대부분의 웹 페이지에서는 UTF-8을 사용하는 것을 권장 -->
<meta charset="utf-8">

<!-- 뷰포트(Viewport 설정) -->
<!-- 모바일 기기에서 웹 페이지가 어떻게 보이는지를 제어하기 위해 사용 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 검색 엔진 최적화(SEO) -->
<meta name="description" content="웹 페이지의 간단한 설명">
<meta name="keywords" content="키워드1, 키워드2, 키워드3">

<!-- http-equiv : 브라우저 호환성 옵션 -->
 <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome">

<!-- refresh : 새로고침하는 주기 설정, content : 주기 값-->
<meta http-equiv="refresh" content="30" />
 <!-- 5초 뒤에 구글 메인으로 이동하는 예시-->
 <meta http-equiv="refresh" content="5, http://www.google.com" />
```

## title
{: .notice--success .text-center}

title 태그는 웹 페이지의 제목을 정의합니다. 웹 브라우저의 탭에 표시되는 텍스트로, 사용자가 현재 열려 있는 페이지를 쉽게 식별할 수 있도록 도와줍니다. 또한 검색 엔진에서도 이 제목을 활용하여 페이지의 콘텐츠를 파악하고 검색 결과에서 제목을 표시합니다.

```html
<!-- title : 상단에 표시될 제목 -->
<title>개발 일지</title>
```

## link
{: .notice--success .text-center}

```html
<!-- 외부 리소스 파일을 연결 (예: CSS 파일, 파비콘 등) -->
<!-- 최소 표현 -->
<link rel="stylesheet" href="main.css">
<!-- 기본 표현 -->
<link rel="stylesheet" href="main.css" media="screen" type="text/css">

<!-- css파일을 link하지 않고 내부 스타일을 직접 작성하는 경우 -->
<style>
    h1 {background-color:coral}
</style>
```

## script
{: .notice--success .text-center}

```html
<!-- 외부 스크립트 파일을 연결 (예 : JS 파일 등) -->
<script type="text/javascript" src="main.js">
    // 내부 스크립트 작성 블록
    window.onload = function() { alert("page loaded"); } // 예시
```

## 텍스트 태그
{: .notice--success .text-center}

```html
<!-- 가로 라인을 만들어주는 태그 -->
<hr>

<!-- 문장의 개행을 지원하는 태그 -->
<br>

<!-- 제목(Heading)을 표시하는 태그 -->
<h1>heading1 입니다.</h1>
<h2>heading2 입니다.</h2>
<h3>heading3 입니다.</h3>
<h4>heading4 입니다.</h4>
<h5>heading5 입니다.</h5>
<h6>heading6 입니다.</h6>

<!-- 긁자 굵기(bold)를 굵게 조절하는 태그 -->
<b>bold 적용</b>

<!-- strong 태그 : bold와 같지만 검색 엔진이나 스크린 리더 등이 인식하여 강조된 텍스트에 더 높은 중요도를 부여함-->
<strong>strong 적용</strong>

<!-- 형광펜 태그 -->
<mark>mark 적용</mark>

<!-- 기울임 태그 : i와 em 모두 기울임, 이탈릭체를 의미하며 em은 추가로 강조된 텍스트의 의미를 부여함 -->
<em>기울임 적용 방법1</em>
<i>기울임 적용 방법2</i>

<!-- 인용 태그 (외부 문자 인용) -->
<!-- q 태그는 짧은 인용문, 보통 문장 내에서 사용하며 더블쿼테이션으로 감싸짐 -->
<q>q 적용</q>
<!-- blockquote 태그는 긴 인용문, 블록 수준의 인용 구간에서 사용되며 들여쓰기된 텍스트로 표시됨 -->
<blockquote>blockquote 적용</blockquote>

<!-- code 태그 : 코드 삽입 태그 -->
<code>
    System.out.println("HelloWorld!");
</code>

<!-- 특수 문자 -->
<!-- 특수문자는 해당되는 코드를 기입해야 함 -->
<!-- 사용 예시 -->
<code>
    &lt;html&gt;&lt;/html&gt;
    &#60;html&#62;&#60;/html&#62;
</code>

<!-- 공백을 연달아 할 경우 한번으로 인식되며 두 번째부터는 &nbsp; 코드를 사용 -->

<!-- 줄바꿈이나 공백의 제한없이 텍스트를 그대로 나타낼 때 사용하는 태그 -->
<pre>       Hello
    World      !!
</pre>
```

## 영역 태그
{: .notice--success .text-center}

```html
<!-- <div>태그는 블록 레벨 요소로, 웹 페이지의 구역(division)을 나눌 때 사용 -->
<!-- 주로 CSS 스타일링이나 JavaScript와 함께 사용되어 일정 영역을 그룹화하고 스타일을 적용하거나 동적인 기능을 추가할 때 유용 -->
<div id="container">
  <p>이 부분은 컨테이너 내의 단락입니다.</p>
  <ul>
    <li>항목 1</li>
    <li>항목 2</li>
  </ul>
</div>

<!-- <span>태그는 인라인 요소로, 특정 부분에 스타일을 적용하거나 JavaScript를 통해 동적인 처리를 할 때 사용 -->
<!-- 주로 텍스트 내의 일부분에 스타일을 적용하고자 할 때 사용 -->
<p>이 문장은 <span style="color: blue;">파란색</span> 글씨를 포함합니다.</p>
```

## 링크 태그
{: .notice--success .text-center}

```html
<!-- <a> 태그는 HTML에서 링크를 생성하는 데 사용되는 태그로, "앵커"(anchor)의 약자입니다. <a> 태그는 다양한 형태의 하이퍼링크를 생성하는 데 활용됩니다. -->
<!-- 외부 링크(절대 경로) -->
<!-- http://, https://를 생략할 경우 내부의 상대 경로로 인식함에 주의 -->
<a href="http://google.com">구글로 이동</a>
<!-- 내부 링크(상대 경로) -->
<a href="00_head.html">00_head.html 로 이동</a>
<!-- target 옵션을 사용해 새로운 탭에서 여는 법 -->
<a href="00_head.html" target="_blank">신규 탭에서 여는 옵션</a>
<!-- 이미지 링크 만들기 -->
<a href="00_head.html">
    <img src="resources/image/flower5.PNG" alt="" height="100px">
</a>
<!-- 다운로드 링크 만들기 -->
<a href="document.pdf" download>Download PDF</a>
<!-- 이메일 링크 만들기 -->
<a href="address@example.com">Contact Us</a>
<!-- 문서 최상단으로 이동 -->
<a href="#top">최상단으로 이동</a>
<!-- 문서 최하단으로 이동 -->
<a href="#bottom">최하단으로 이동</a>
<!-- 최하단에 해당 코드 작성 -->
<div id="bottom" class="bottomClass"></div>
```

## form 태그
{: .notice--success .text-center}

form 태그는 HTML에서 사용자로부터 정보를 수집하고 서버로 전송하는 데 사용되는 중요한 요소입니다. 웹 양식(HTML Form)은 사용자 입력을 받는 데 사용되며, 다양한 유형의 입력 필드, 버튼, 체크박스, 라디오 버튼 등을 포함할 수 있습니다.

```html
<!-- action : form이 전송될 URL을 지정합니다. 사용자가 폼을 제출할 때 데이터가 이 URL로 전송됩니다. -->
<!-- method : get과 post는 데이터를 서버로 전송하는 데 사용되는 두 가지 주요 HTTP 메소드입니다. get은 주로 데이터를 URL의 쿼리 문자열(query string)에 포함시켜 서버로 전송하고 주로 검색 결과, 필터링 옵션 등을 서버에 요청할 때 사용됩니다. post는 데이터를 HTTP 요청의 본문(body)에 포함시켜 서버로 전송하고 주로 로그인 정보, 회원가입 양식, 파일 업로드 등과 같이 데이터가 민감한 경우나 길이가 긴 데이터를 전송할 때 사용됩니다. 일반적으로, GET은 데이터를 요청할 때, POST는 데이터를 서버로 제출할 때 사용됩니다. -->
<!-- name : 해당 폼을 서버로 제출할 때 특정 이름을 지정하여 데이터를 전송할 수 있습니다. 이는 여러 폼이 한 페이지에 있을 때, 어떤 폼에서 데이터가 전송되었는지를 서버에서 구분할 때 사용될 수 있습니다.  -->
<form action="00_head.html" method="get" name="firstForm">
    <!-- <fieldset> 요소는 HTML에서 여러 관련된 입력 필드들을 그룹화하고 그룹에 대한 설명을 제공하는 데 사용되며 주로 해당 그룹의 제목을 제공하는 <legend> 요소와 함께 사용됩니다. -->
    <fieldset>
        <legend>회원</legend>
        <div>
            <!-- for와 id의 값은 일치시켜야 하며 중복될 수 없고 name은 서버에서 활용하며 id와 일치시키는게 보편적입니다. -->
            <!-- required 속성을 사용하면 브라우저는 사용자가 폼을 제출할 때 필수 입력 필드가 비어 있으면 경고를 표시하고 폼이 제출되지 않도록 막습니다. -->
            <label for="memberID">아이디</label>
            <input type="text" id="memberID" name="memberID" required>
        </div>
        <div>
            <!-- minlength, maxlength 속성을 사용해 해당 필드의 최소, 최댓값을 정할 수 있습니다. -->
            <!-- placeholder를 통해 해당 필드가 비었을 때 사용자가 어떤 정보를 기입해야 하는지에 대한 도움말을 제공합니다. -->
            <label for="pwd">비밀번호</label>
            <input type="password" id="pwd" name="pwd" minlength="8" maxlength="30" placeholder="8자 이상 30자 이하" required>
        </div>
        <div>
            <!-- value 속성을 사용해 필드의 초기값을 지정할 수 있습니다. -->
            <!-- number 타입은 min, max를 통해 최소, 최대값 설정 -->
            <label for="age">나이</label>
            <input type="number" id="age" name="age" value="20" min="20" max="60">
            <!-- 이 외에 readonly 속성을 통해 필드를 읽기 전용으로 만들 수 있습니다. -->
        </div>
    </fieldset>
    <fieldset>
        <legend>기타정보</legend>
        <div>
            <label for="email">email</label>
            <input type="email" id="email" name="email">
        </div>
        <div>
            <label for="homepage">홈페이지</label>
            <input type="url" id="homepage" name="homepage">
        </div>
        <div>
            <label for="phone">전화번호</label>
            <input type="tel" id="phone" name="phone">
        </div>
    </fieldset>
    <fieldset>
        <legend>프로필 소개</legend>
        <textarea name="info" id="info" cols="50" rows="10" style="resize:none;"></textarea>
    </fieldset>
    <fieldset>
        <legend>숫자 입력 관련 태그</legend>
        <div>
            <input type="number" name="point" id="point" max="100" min="0">
        </div>
        <div>
            <input type="range" name="point2" id="point2" max="100" min="0">
        </div>
    </fieldset>
    <fieldset>
        <legend>날짜/시간 관련 태그</legend>
        <div>
            <label for="birthday">생일</label>
            <input type="date" name="birthday" id="birthday" value="2000-10-10">
        </div>
        <div>
            <label for="reserved">시간 예약</label>
            <input type="time" name="reserved" id="reserved" value="14:20">
        </div>
        <div>
            <label for="reserved2">날짜/시간 예약</label>
            <input type="datetime-local" name="reserved2" id="reserved2" value="2023-12-06T15:39">
        </div>
        <!-- date : 년, 월, 일 선택 가능한 양식 제공 -->
        <!-- <input type="date" name="date2" min="2022-01-01" max="2022-12-31" value="2022-06-10"> -->
        <!-- datetime-local : 년, 월, 일, 시간 선택 가능한 양식 제공 -->
        <!-- <input type="datetime-local" name="date3"> -->
        <!-- month : 년, 월 선택 가능한 양식 제공 -->
        <!-- <input type="month" name="date4"> -->
        <!-- week : 주 선택 가능한 양식 제공 -->
        <!-- <input type="week" name="date5"> -->
        <!-- time : 시간만 선택 가능한 양식 제공 -->
        <!-- <input type="time" name="date6"> -->
    </fieldset>
    <fieldset>
        <legend>선택 관련 태그</legend>
        <!-- 라디오 -->
        <!--name : 라디오를 그룹핑할 네임으로 중복해서 사용-->
        <!--id : 각자 선택된 값으로 부터 다른 id 부여-->
        <!--value : 실제 서버로 넘어갈 값-->
        <input type="radio" name="gender" id="gender-male" value="m" checked>
        <label for="gender-male">남</label>
        <input type="radio" name="gender" id="gender-female" value="f" >
        <label for="gender-female">여</label>
        <!-- 체크박스 -->
        <!--name : 체크박스를 그룹핑할 네임으로 중복해서 사용-->
        <!--id : 각자 선택된 값으로 부터 다른 id 부여-->
        <!--value : 실제 서버로 넘어갈 값-->
        <!--주의점 : 서버에서는 배열로 처리해야 다중 선택을 처리할수 있다.(서버에서 다루기 까다로움) -->
        <input type="checkbox" name="hobby" id="music" value="music" checked>
        <label for="music">음악감상</label>
        <input type="checkbox" name="hobby" id="game" value="게임" >
        <label for="game">게임</label>
        <input type="checkbox" name="hobby" id="book" value="book" >
        <label for="book">독서</label>
        <input type="checkbox" name="hobby" id="health" value="health" checked>
        <label for="health">헬스</label>
        <!-- 단일 사용 설계, value가 없음 -->
        <label for="checkmail">메일 수신</label>
        <input type="checkbox" name="checkmail" id="checkmail">
        <!-- select -->
        <!-- size 옵션을 활용해 화면에 보여줄 개수를 조절할 수 있음 -->
        <label for="nation">국가</label>
        <!-- 가장 기본적인 형태, value가 없는 option, 표현값으로 value 대체됨-->

            <optgroup label="America">

            </optgroup>

        <!-- datalist -->
        <label for="menu">과일 선택</label>
        <input type="text" name="menu" id="menu" list="menu-list">
        <datalist id="menu-list">
            <option value="apple">사과</option>

        <!-- 컬러 선택 -->
        <label for="userColor">컬러 : </label>

        <label for="upFile">파일 업로드</label>

        <!-- hidden -->
        <input type="hidden" name="hiddenValue" value="안보이지만 서버로 전송되는 데이터!">
        <input type="text" name="hiddenValue2" value="보이는 정보!">
        <!-- search 검색어 입력 양식  -->
        <input type="search" name="search">
        <!-- prograss : 작업 진행 상태를 나타내는 태그 -->
        <progress max="100" value="70"></progress>
        <!-- meter : 값이 차지하는 크기(게이지) 표시 ex) 디스크 사용량 등 -->
        <meter min="0" max="10" low="3" high="7" value="5"></meter>
    </fieldset>
    <div>
        <input type="submit" value="제출">
        <input type="reset" value="리셋">
        <!-- 버튼은 input 태그로 사용하지 않고 button 태그로 사용 권장 -->
        <input type="button" value="버튼">
        <!-- button -->
        <button type="submit" value="제출">
        <!-- onclick 속성을 통해 이벤트 동작 가능 -->
        <button type="button" onclick="alert('button test');">
            <!-- 이미지 버튼 만드는 법 -->
            <img src="test.png">
        </button>
        <!-- input을 이용해 이미지 버튼 만드는 법 -->
        <input type="button" value="버튼" onclick="javascript:alert('Hello');">
    </div>
</form>
```

## iframe 태그
{: .notice--success .text-center}

HTML 문서 내 또 다른 HTML 문서를 삽입하는 태그, 외부 페이지를 내부에 가져오는 태그로 주로 배너광고, 일부 페이지를 따올 때나 서버와 다른 서버 페이지를 연동해야할때 사용, ex: 스프링 + django

```html
<!-- 경로를 직접 지정해서 불러오는 방법 -->
<iframe src="test.html" width="100%"></iframe>

<!-- a링크를 통해서 프레임을 선택하는 방법 -->
<a href="test1.html" target="f1">html 1</a>
<a href="test2.html" target="f1">html 2</a>
<iframe name="f1" width="100%"></iframe>

<!-- 프레임의 크기 및 스타일 속성 -->
<iframe src="test.html" width="100%" height="500px" style="border: none;"></iframe>
```
